# fiap-tech-challenge-fase-2
O sistema hospitalar enfrenta desafios logísticos na distribuição de insumos entre unidades e para atendimento  domiciliar. O desafio é desenvolver um sistema de otimização de rotas  utilizando algoritmos genéticos para resolver esse problema de caixeiro viajante médico, utilizar LLMs para gerar relatórios e instruções para as equipes de entrega.
